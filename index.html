What is MINX?

  • MINX is: A C++ Game Development Framework
  • MINX is: A Rapid Prototyping Tool for Creating C++ Data Visualizations
  • MINX Is: Not XNA

How do I use MINX?

On GNU/Linux

Clone our repository, or grab a snapshot from above. Then run the following commands.

  1. sudo ./configure
  2. make
  3. sudo make install
  4. sudo make headers

After that you should have MINX fully installed. Head over to the TestMINXProject page to confirm!
